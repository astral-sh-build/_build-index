Astral GPU indexes

Pre-built, GPU-enabled wheels for the Python ecosystem.

Pre-built wheels for popular GPU-enabled packages in the PyTorch ecosystem, like FlashAttention, across a range of Python versions, CPU architectures, CUDA versions, and PyTorch versions, based on open source build pipelines.

Following the PyTorch convention, the Astral GPU indexes use a separate index URL for each supported CUDA version (e.g., /simple/cu128), and publish a separate wheel version for each supported CUDA and PyTorch combination (e.g., +cu.12.8.torch.2.11).
